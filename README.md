- VEDA  AI Powered Student Learning Companion 🎓🤖

VEDA is an AI-driven learning companion designed to go beyond traditional marks-based evaluation systems.  
It analyzes *how students learn* by assessing key learning fundamentals and provides personalized feedback to improve academic performance.

---

 🚀 Problem Statement

Students in the same class often score differently despite learning the same syllabus.  
Traditional evaluation systems:
- Focus only on marks and grades
- Fail to explain *why* performance differs
- Ignore core learning abilities

  Key learning gaps identified:
- 🎧 Listening Skills
- 🧩 Grasping Power
- 🧠 Retention Power
- ✍️ Application Ability

---
 💡 Solution – VEDA

VEDA uses AI to create personalized learning journeys by:
- Generating adaptive quizzes
- Analyzing student responses
- Providing actionable feedback on learning fundamentals
- Helping students, teachers, and parents understand performance gaps

---

✨ Features

- 🎯 **Class, Subject & Difficulty Selection**
- 📊 **AI Feedback Dashboard** with radar charts
- 🤖 **AI Tutor (Mascot-based Chatbot)**
- 👨‍👩‍👧 **Multiple User Roles** (Student, Teacher, Parent)
- 🌗 **Gamified & Student-Friendly UI** (Light/Dark Mode)
- 📈 **Performance Tracking & Progress Monitoring**

---

 🧠 How It Works

1. Student selects class, subject, and difficulty
2. AI generates a customized quiz
3. Student attempts the quiz
4. AI analyzes responses
5. Feedback is generated on learning fundamentals
6. Difficulty level is adjusted automatically
7. Student retries with improved guidance
